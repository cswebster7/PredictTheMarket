Pre-requisite- "npm" and "babel"

Steps:
1) Install react by : $npm install react --save followed by $ npm install react-dom --save.
2) Clone the repository: $ git clone -b cognito https://github.com/cswebster7/PredictTheMarket.git
3) Go inside PredictTheMarket folder and run: a)$ npm init (press enter at each option)
					      b)$ npm install
					      c)$ npm install --save aws-amplify
					      d)$ npm install --save universal-cookie
					      e)$ npm start (This command will start the development server at localhost on default browser)

			****Install any other package if required****
**** If npm start command gives any error then close the browser and terminal, start a new terminal and run npm install again followed by npm start.
